 This archive contains the last release of the CAEN HV Control Software and the
 corresponding Release Notes.


 Content of the archive
 ----------------------

 CAENHVControlSoftwareSetup.exe :  The setup file that install the CAEN HV Control Software on the pc

 Readme.txt				        :  This file
 ReleaseNotes.txt				:  Release Notes of the last software release


 System Requirements
 -------------------
 
 - CAEN HV Power Supply System SY4527 / SY5527, V65XX, NIM /DESKTOP HV
 - Windows Xp/Vista/7 32bit and 64bit
 - Linux 32bit and 64bit
 - glibc 2.9 or higher

 Installation notes
 ------------------

 - Launch the setup program.
 - Follow the instructions of the setup program.
 - Once setup process has finished, the CAEN HV Control Software is 
   installed on your pc.
